# Ejercicios de Java para Practicar

Este repositorio es una colección personal de ejercicios básicos de Java, diseñados para practicar y afianzar los fundamentos de la programación.

## Propósito

El objetivo principal de este proyecto es servir como un campo de práctica para resolver problemas comunes y mejorar las habilidades en Java. Los ejercicios aquí presentes son una mezcla de diferentes fuentes y no siguen un orden o temario específico, sino que se van añadiendo a medida que se encuentran y resuelven.

## Contenido

Los ejercicios cubren una variedad de temas fundamentales, incluyendo:

*   **Operaciones matemáticas y lógicas.**
*   **Manejo de variables y tipos de datos.**
*   **Estructuras de control (condicionales y bucles).**
*   **Entrada y salida de datos por consola.**
*   **Resolución de problemas algorítmicos simples.**

Cada fichero `.java` en el directorio `src` corresponde a un ejercicio individual y es autocontenido.
